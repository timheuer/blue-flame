Firebase tools in VS Code

Blue Flame is a Visual Studio Code extension for browsing and managing Google Cloud Firestore, Firebase Authentication users, and Firebase Storage files.

VS Code Marketplace GitHub Repository

Features

  • Browse Firestore collections, documents, and subcollections in a tree view.
  • Open collection tables and edit JSON documents directly in VS Code.
  • Manage Firebase Authentication users with create, edit, disable, and delete operations.
  • Browse and preview Firebase Storage files with metadata and copyable URLs.
  • Use Google OAuth, Application Default Credentials, or service account authentication.
